# Slothello

Slothello on versio othello nimisest‰ kaksinpelist‰, jossa tarkoituksena on saada mahdollisimman monta omaa nappulaa ruudukolle.
Pelaajat vuorotellen laittavat nappulan ruudulle, siten ett‰ j‰tt‰v‰t uuden nappulan ja jonkun oman nappulan v‰liin toisen pelaajan napin,
ja k‰‰nt‰v‰t v‰liin j‰‰neet nappulat. Peli loppuu kun ruudukko on t‰ynn‰ tai lailliset siirrot loppuvat
